# FieldsSelectorUnion

An object to specify a list of field names to include and/or exclude in the result.


## Supported Types

### `models.FieldsSelector1`

```typescript
const value: models.FieldsSelector1 = {
  include: [
    "<value 1>",
  ],
};
```

### `models.FieldsSelector2`

```typescript
const value: models.FieldsSelector2 = {
  exclude: [
    "<value 1>",
  ],
};
```
